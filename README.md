ng2seed v0.1
====================

An Angular 2 seed.

+ Framework/Library: AngularJS, Traceur
+ Language: TypeScript
+ Template: Jade
+ Dependency and Package Management: NPM, BOwer
+ Task Runner: Grunt
+ Test Frameworks: PhantomJS, Karma, Jasmine, Protractor


Installation:
-----------------

<code>$ npm install</code><br>
<code>$ jspm install</code>


Tests:
-----------------

<code>$ grunt tests</code>


Run:
-----------------

<code>$ grunt web</code>
